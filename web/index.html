Combinatorial Game Theory (CGT)

This site is a browser-based platform for computing values of short combinatorial games. The calculations run locally through WebAssembly compiled from the C core.

Available Modules

  • Hotpotch: A colored-edge game with blue, red, green edges and gravity.
  • Domineering: Vertical and horizontal domino placement.
  • Toads & Frogs: Toads move right and frogs move left.
  • JS Sandbox: Write custom game rules in JavaScript using the object API.
